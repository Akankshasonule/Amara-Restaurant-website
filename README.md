# Amara-Restaurant-website
# using Frontend: React.js, HTML, CSS, JavaScript

**Amara Restaurant Website** is a modern and responsive restaurant website built using **HTML, CSS, JavaScript, and React.js**
It provides a seamless user experience with beautiful UI, menu display, and easy navigation.

# Features
Fully Responsive (Mobile & Desktop friendly)
Modern UI/UX Design
Interactive Menu Section
Smooth Animations & Transitions
Reservation Booking Section 
Optimized Performance

# Clone the repository
git clone https://github.com/Akankshasonule/Amara-Restaurant-website.git

# Navigate to the project folder
cd Amara-Restaurant-website

# Install dependencies
npm install

# Start the development server
npm start
